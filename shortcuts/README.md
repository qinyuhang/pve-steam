# iOS 快捷指令

## 方法一：网页配置（推荐）

1. 编辑 `import.html`，填入你的 PVE IP、端口和 Token
2. 把 `import.html` 放到内网 Web 服务器，或传到手机
3. iPhone 用 Safari 打开，点击按钮即可复制 URL
4. 按页面教程创建快捷指令

## 方法二：手动创建

### 1. 创建"开电脑"快捷指令

打开 iOS **快捷指令 App**，新建快捷指令：

1. **添加操作** → **网络** → **获取 URL 内容**
2. 方法选择 **GET**
3. URL 填入：`http://你的PVE_IP:8080/start?token=你的TOKEN`
4. **添加操作** → **脚本** → **显示通知**
   - 标题："开电脑"
   - 内容："正在启动 VM..."
5. 点击右上角 **设置** → **名称**："开电脑"
6. **添加到主屏幕** → 选择图标

### 2. 创建"关电脑"快捷指令

同上，URL 改为：`http://你的PVE_IP:8080/stop?token=你的TOKEN`

### 3. 添加 Siri 语音控制

在快捷指令设置中点击 **"添加到 Siri"**，录制语音指令：
- "开电脑"
- "关电脑" 
- "启动 Steam"

然后就可以说：
> "嘿 Siri，开电脑"

## 方法三：生成脚本

使用项目提供的脚本自动生成配置：

```bash
cd shortcuts
./generate_shortcut.sh 192.168.1.100 8080 your-token
```

脚本会输出：
- URL Scheme（尝试自动跳转快捷指令 App）
- 二维码（扫码访问 URL）
- 手动配置说明

## 快捷指令效果

创建完成后，你可以：

| 方式 | 操作 |
|------|------|
| 主屏幕图标 | 点击直接开电脑 |
| Siri 语音 | "嘿 Siri，开电脑" |
| 自动化 | 连接家里 WiFi 时弹出快捷按钮 |
| 控制中心 | 添加到控制中心快速开关 |

## 常见问题

**Q: iOS 提示无法验证快捷指令？**
A: 在 设置 → 快捷指令 → 允许不受信任的快捷指令 中开启

**Q: 快捷指令运行没反应？**
A: 检查：1) 手机和 PVE 是否在同一局域网 2) Token 是否正确 3) 防火墙是否放行端口

**Q: 可以添加到锁屏界面吗？**
A: iOS 16+ 支持锁屏小组件，可以将快捷指令添加到锁屏
